# 玩测说明

## 试玩链接

### 本机试玩
```powershell
cd E:\OpenCode\Game_01
npx vite
```
浏览器打开 http://localhost:5173

### 局域网手机试玩
```powershell
npx vite --host
```
终端会输出 Network 地址（如 http://192.168.x.x:5173），手机同 Wi-Fi 打开即可。

### 在线试玩（如需外网访问）
可推送到 GitHub 临时分支获取 Pages 链接，或 itch.io 草稿页（T8/T9 阶段）。

## 玩法说明（给试玩者）

1. 按空格或点击屏幕开始
2. 选择角色：
   - **守序者（小蟾蜍）**：把变乱的东西归位——在正确时机按键，让东西"啪"地回到该在的位置
   - **闯入者（候鸟）**：别打乱想融入——在正确时机按键，小心翼翼地不碰乱整齐的东西
3. 按错没关系，热闹不惩罚，继续玩
4. 归位/融入 10 个东西后进入结局

## 无解说原则

不要教试玩者怎么玩，只给上面这段说明。观察：
- 第一次按键是什么时候？
- 什么时候笑了/困惑了/卡住了？
- 是否想再玩一次？
- 手机端触摸是否顺畅？
