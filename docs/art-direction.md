# 视觉与音频母题 — Art Direction

> 记录本游戏的视觉风格、配色、字体、音频气质和资产使用规则。

## 视觉母题

### 核心对比：秩序几何 vs 有机暖色

- **守序者世界** = 几何对称、冷色系（青 #4ecdc4）、精确网格、直线
- **闯入者元素** = 有机曲线、暖色系（橙红 #ff6b6b）、打破网格、弧线
- 两者碰撞 = 画面本身在讲"秩序 vs 混乱"的故事

### 配色（≤4 色）

| 色值 | 用途 | 角色 |
|---|---|---|
| #1a1a2e | 背景（深蓝紫） | 中性 |
| #4ecdc4 | 秩序/守序者/命中 | 冷色 |
| #ff6b6b | 混乱/闯入者/失败 | 暖色 |
| #ffffff | 文字、高亮、高光 | 中性 |
| #ffd700 | Combo、对勾、奖励 | 强调（金色） |

### 场景设计

- **房间**：深蓝紫墙壁 + 深蓝地板，营造夜晚/室内的温馨感
- **书架/柜子**：棕色（#8B7355），位于屏幕右侧（守序者）或左侧（闯入者）
- **物品**：6 种不同形状和颜色，代表日常生活中要被归位的物件

### 物品图鉴

| 物品 | 形状 | 尺寸 | 代表意象 |
|---|---|---|---|
| 书 | 矩形 | 18×24 | 知识/秩序 |
| 杯子 | 圆形 | r=10 | 日常/温暖 |
| 袜子 | 矩形 | 12×16 | 琐碎/凌乱 |
| 相框 | 矩形 | 20×20 | 记忆/珍贵 |
| 花瓶 | 圆形 | r=8 | 美感/易碎 |
| 闹钟 | 圆形 | r=12 | 时间/精确 |

### 动画风格

- **成功**：金色粒子爆炸 + 对勾 + 浮动文字（"+10" / "3x COMBO!"）
- **失败**：物品飞走/掉落 + 旋转 + 粒子散射（暖红色）
- **结局**：淡入动画，角色几何化简呈现递东西动作

## 音频母题

### 气质对比：精准节拍 vs 温暖即兴

- **守序者**：整齐节拍（类似节拍器滴答），简约重复
- **闯入者**：不规则节奏，即兴感
- **实际实现**：统一使用 Web Audio API 合成，通过频率和节奏差异体现角色差异

### 音效设计

| 事件 | 音色 | 参数 |
|---|---|---|
| 命中 | 高音"叮" | sine, 440Hz + combo*50, 0.15s |
| 连击加成 | 和声"叮-叮" | 主音 + 五度音 |
| 失败 | 低音"哐" | sawtooth, 150Hz, 0.3s |
| 错过 | 沉闷"嗡" | square, 100Hz, 0.2s |
| 节拍提示 | 轻"嗒" | sine, 200Hz, 0.05s, 低音量 |

### 音频策略

- **首次交互后初始化**：遵守浏览器自动播放策略
- **不引入外部音频资产**：全部用 Web Audio API 合成，零外部依赖
- **可选扩展**：若合成音色不足，可引入 CC0 音效（须写 ADR）

## 字体

- **主字体**：system-ui, -apple-system, sans-serif
- **风格**：无衬线，清晰易读，不引入外部字体文件

## 触摸友好

- **点击区域**：全屏 canvas 均为点击区域，无需精确点击小目标
- **按钮尺寸**：角色选择按钮 ≥100px，远大于 44px 最低标准
- **安全区域**：viewport-fit=cover + env(safe-area-inset-*) 适配刘海屏
- **横竖屏**：响应式 Canvas，内容随短边缩放
